# Java-Innovative-Projects
A collection of Java projects including a Password Generator, Expense Tracker, and Text Steganography System.
Projects
1. Smart Password Generator
Generates passwords using Name, PAN Number, and Date of Birth.
Uses SecureRandom for random numbers and special characters.
Checks password strength (Weak, Medium, Strong).
2. Personal Expense Tracker
Add and display expenses.
Generate monthly expense reports.
Find the highest expense category.
Save and load expense records using serialization.
3. Text Steganography
Hides secret messages inside a text file.
Uses invisible Unicode characters for encoding.
Retrieves the hidden message from the encoded file.
